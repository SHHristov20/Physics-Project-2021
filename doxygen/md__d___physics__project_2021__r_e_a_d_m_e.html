『🕹』 Deploying

To deploy our project just follow these steps:

  1. Clone it. This can be done by: 
    git clone https://github.com/SHHristov20/Physics-Project-2021
  2. Navigate to the Project directory: 
    cd Physics-Project-2021
  3. Navigate to the debug folder: 
    cd application/Debug
  4. Run the .exe file by typing: 
    PhysicsProject.exe

『🎮』Controls

W - Move the pointer one block up

A - Move the pointer one block to the right

S - Move the pointer one block down

D - Move the pointer one block to the right
